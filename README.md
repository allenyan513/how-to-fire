# How to FIRE

一本面向普通人的中文 FIRE 开源书：把"财富、复利、长期主义和提前退休"这些事，用尽量朴素的方式讲清楚。

这不是一本暴富指南，也不是投资荐股书。它更像作者在工作、消费、储蓄、投资和人生规划中，逐渐沉淀下来的一份认知笔记：我们如何更早理解钱的运行规律，如何避免被短期情绪牵着走，如何把生活从"只靠工资"慢慢调整为"有系统地积累资产"。

写它，本身也是作者对这些概念的一次重新整理。

## 这本书想讲什么

- 高收入不等于财富积累
- 劳动收入和资产收入是两套系统
- 复利真正重要的是时间，而不是收益率
- FIRE 不是逃离工作，而是获得更多人生选择权
- 普通人的财富系统，来自现金流、储蓄率、长期投资、风险控制和稳定心态

## 谁适合读

- 想开始理解财富和提前退休，但不想被复杂金融术语劝退的人
- 收入还可以，却总觉得资产积累很慢的人
- 对复利、指数基金、资产配置、4% 法则等概念感兴趣的人
- 想建立长期主义生活方式，而不是追逐短期暴富故事的人

## 这本书不是什么

- 不是荐股书，不推荐具体买卖。
- 不是暴富书，不承诺收益。
- 不是极端节俭书，不把降低生活质量当成目标。
- 不是个人化财务规划，不能替代专业建议。
