# How to FIRE

一个面向普通人的中文公开课项目，讨论如何理解财富、复利、长期主义和提前退休。

这不是一套教人暴富的课程，也不是投资荐股课。它更像是一份普通人在工作、消费、储蓄、投资和人生规划中逐渐形成的认知笔记：我们如何更早理解钱的运行规律，如何避免被短期情绪牵着走，如何把生活从“只靠工资”慢慢调整为“有系统地积累资产”。

## 课程定位

本项目第一阶段以公开课内容仓库为主，先把课程大纲、讲稿、案例和课后思考整理清楚。

核心主题：

- 高收入不等于财富积累
- 劳动收入和资产收入是两套系统
- 复利真正重要的是时间
- FIRE 不是逃离工作，而是获得更多人生选择权
- 普通人的财富系统，来自现金流、储蓄率、长期投资、风险控制和稳定心态

## 适合谁

- 想开始理解财富和提前退休，但不想被复杂金融术语劝退的人
- 收入还可以，却总觉得资产积累很慢的人
- 对复利、指数基金、资产配置、4% 法则等概念感兴趣的人
- 想建立长期主义生活方式，而不是追逐短期暴富故事的人
