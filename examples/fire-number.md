# 年度支出、FIRE 数字和 4% 法则

这个示例用来理解 FIRE 数字的粗略估算方式。它不是退休建议，也不是财务规划。

## 简化公式

如果使用 4% 法则做非常粗略的估算：

```text
FIRE 数字 = 年度支出 ÷ 4%
```

也可以理解为：

```text
FIRE 数字 = 年度支出 × 25
```

## 结果示例

| 年度支出 | 粗略 FIRE 数字 |
| ---: | ---: |
| 120000 | 3000000 |
| 180000 | 4500000 |
| 240000 | 6000000 |
| 360000 | 9000000 |
| 480000 | 12000000 |

## 如何理解

这个表格最重要的启发，不是让人记住某个固定数字，而是看见两个事实：

- 年度支出越高，需要支撑它的资产规模越大。
- FIRE 不是只看资产，也要看生活方式和现金流。

同样的资产规模，对低固定支出的人可能很宽松，对高固定支出的人可能仍然紧张。

## 4% 法则的边界

4% 法则只是一个常见思考框架，不是保证。

它会受到很多因素影响：

- 退休年限
- 资产配置
- 市场波动
- 通胀
- 税务
- 医疗和家庭责任
- 所在国家和地区
- 是否有养老金、兼职收入或其他现金流

## 关键提醒

- 提前退休通常比传统退休需要更长的资金支持时间。
- FIRE 计划应该保守、有弹性，而不是只追求一个漂亮数字。
- 真正重要的是现金流能否长期支持生活，而不是单纯拥有一个资产总额。
