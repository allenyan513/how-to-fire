# 第二部分 · 财富工具

第一部分讲的是"为什么"。这一部分讲"是什么"——

> 世界上的财富工具各是什么、为什么会存在？

很多人接触金融的第一步，是被各种术语劝退的：tracking error、expense ratio、beta、共识算法、哈希、布雷顿森林⋯⋯听上去像另一个语言。这一部分用普通人的语言，重新讲一遍那些常被听到的概念：股票、ETF、指数基金、黄金、比特币、国债、房产、现金流、4% 法则、资产配置。

切入角度是统一的——**它们为什么存在**——而不是"现在应该买什么"。这一部分不教你交易，它帮你建立一张关于财富世界的基本地图。
