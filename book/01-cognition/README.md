# 第一部分 · 财富认知

这一部分回答一个最基础的问题——

> 为什么很多人努力工作很多年，收入并不低，却始终没有真正的自由感？

答案不在于赚得不够多，而在于对"钱"和"时间"的直觉本身就是错位的：我们容易把收入当成财富，把复利当成利率，把波动当成风险，把消费升级当成生活变好。在讨论"做什么"和"怎么做"之前，先得把这些直觉一节一节拆开看。

本部分共 8 节，每一节都是一次独立的认知矫正。
